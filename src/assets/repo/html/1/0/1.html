For positive integers j and k

  • R(j,k) = n iff every graph with n or more vertices has either a j clique or an independent set of size k
  • A (j,k;n,e) graph is a graph of n vertices and e edges in which there is no j clique or set of k independent vertices

    a (j,k) graph is a graph with no j clique or set of k independent vertices

    a (j,k;n) graph is a graph with n vertices in which there is no j clique or set of k independent vertices

    a (j,k;n,e) graph is a graph with n vertices in which there is no j clique or set of k independent vertices with e edges

  • e(j,k,n) is the minimum number of edges in any (j,k;n) graph
