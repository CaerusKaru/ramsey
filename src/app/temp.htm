Introduction

Explanation of Ramsey numbers.

Derive upper and lower bounds for R(3,10). For a overview of the proof technique used to derive the upper bounds, see our proof that R(3,4) is less than 10.

Read as preparation for reading source material, which includes more results.

Definitions

Graphs

  • For any graph G
    • V (G) is the vertex set of G
    • E(G) is the edge set of G
    • e(G) is the number of edges in G
    • For any vertex v of G
      • N(v) (or Nv(G)) is the set of neighbors of v
      • degv(G) is the degree of v
    • α(G) is the size of the largest independent set in G

      (or, its independence number)

    • δ(G) is the minimum degree of any vertex in G
    • Δ(G) is the maximum degree of any vertex in G

Ramsey numbers

  • For positive integers j and k
    • R(j,k) = n iff every graph with n or more vertices has either a j clique or an independent set of size k
    • A (j,k;n,e) graph is a graph of n vertices and e edges in which there is no j clique or set of k independent vertices

      a (j,k) graph is a graph with no j clique or set of k independent vertices

      a (j,k;n) graph is a graph with n vertices in which there is no j clique or set of k independent vertices

    • (j,k;n,e) is the set of all (j,k;n,e) graphs
    • Any (j,k;R(j,k) - 1) graph is critical for (j,k)

      That is, in order to establish that R(j,k) = m, we must construct a (j,k;m - 1) graph (as well as prove that R(j,k) < m + 1)

Minimal edges/graph structure

  • e(j,k,n) is the minimum number of edges in any (j,k;n) graph

    (or ∞ if no such graph exists)

  • Z(v) (or ZG(v)) is the sum of the degrees of all neighbors of v in G

    Z(v) = ZG(v) = ∑ {u,v}∈E(G)deg(v)

  • If G ∈ (3,k), then Δ(G) < k

    because all neighborhoods of G are independent

  • For any (3,k;n,e) graph G
    • Gv is the graph induced in G by the set V (G) - (NG(v) ∪{v})
    • γ(v) is the deficiency of vertex v

      γ(v) = e - Z(v) - e(3,k - 1;n - deg(v) - 1) ≥ 0

      In other words, the number of edges in the induced graph Gv are at least as large as the minimum number of edges in a graph of that type

    • γ(G) is the deficiency of the graph

      γ(G) = ∑ v∈V (G)γ(v)

There is a single (3,9;35) graph, and it is 8-regular with 140 edges

Since R(3,9) = 36, we know there is some (3,9;35) graph, call it G. The most edges G can contain is 140: if there were more, one vertex would have to be degree 9. This would entail an independent set of size 9, since G is triangle free. So G ∈ (3,9;35,140 - s) for some s ≥ 0.

We now show that δ(G) > 6. Suppose there were some vertex v with degree 6 (or less). Then Gv is a (3,8;≥ 28) graph. But R(3,8) = 28, so this cannot be the case.

So every vertex in G is either degree 7 or 8. Let there be n7 degree 7 vertices, and n8 degree 8 vertices. We show that n7 = 2s. Since every edge is degree 7 or 8, the only way to have 140 edges is to have all vertices of degree 8. If G has fewer than 140 edges, some of these edges must be missing, and every missing edge is incident to 2 vertices, so n7 = 2s.

Solving for (4), the only values for which γ(G) is non-negative are s = 0,1,2,3,4. This shows that G contains at least 136 edges.

Let H be the graph induced in G by all n7 vertices of degree 7. Since H has 2s vertices, δ(H) ≤ s. Therefore, in G, any degree 7 vertex is adjacent to at most s other degree 7 vertices.

Suppose, for the sake of contradiction, that n7 > 0 (or, equivalently, that s > 0). Let v be some vertex of degree 7. Then Z(v) ≥ 7s + 8(7 - s) = 56 - s, where s is the number of degree 7 vertices adjacent to v, and 7 - s the number of degree 8 vertices adjacent to v. Also, the number of edges in Gv ≤ (140 - s) - (56 - s) = 84. However, Gv is a (3,8;27) graph and e(3,8,27) = 85 (found using mtf method). Therefore, n7 = 0, i.e. every vertex in G is degree 8, and G contains 140 edges.

For any vertex v in G, Gv is a (3,8;26,76) graph. Using neighborhood gluing, the authors reconstruct every (3,9;35;140) graph from each (3,8;26,76) graph (found using mtf method), and show that all constructed graphs are isomorphic. Therefore, there is a unique (up to isomorphism) 8-regular (3,9;35) graph, and it contains 140 edges.

The graph is a cyclic graph on 35 vertices with circular distances {1,7,11,16}.
